# Introduction to MLTrack

MLTrack is a powerful enhancement for MLflow that adds deployment capabilities, cost tracking, and a modern UI while maintaining 100% compatibility with your existing MLflow code.

## Why MLTrack?

- Writing custom deployment scripts
- Building your own UI
- Manually tracking costs

## How It Works

MLTrack acts as a layer on top of MLflow:

```python
# Your existing MLflow code continues to work
import mlflow
mlflow.start_run()
mlflow.log_metric("accuracy", 0.95)

# Add MLTrack for superpowers
from mltrack import ship
ship("my-model", platform="modal")
```

## Key Concepts

Before diving in, it's helpful to understand these key concepts:

- **Experiments** - Groups of related model runs
- **Runs** - Individual model training sessions
- **Metrics** - Quantitative measurements (accuracy, loss, etc.)
- **Parameters** - Model configuration (learning rate, batch size, etc.)
- **Artifacts** - Files produced by runs (models, plots, etc.)
- **Deployments** - Production-ready model endpoints
