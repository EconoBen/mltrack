## Usage

To use the design system in your documentation:

1. Import the CSS files:
```css
@import '/src/styles/design-system.css';
@import '/src/styles/components.css';
@import '/src/styles/typography.css';
```

2. Use the CSS variables:
```css
.my-element {
  color: var(--color-primary-500);
  padding: var(--space-4);
  font-size: var(--text-lg);
}
```

3. Apply component classes:
```html
<button class="btn btn-md btn-primary">
  Click me
</button>
```

## Design Principles

1. **Clarity First**: Clear hierarchy and scannable content
2. **Performance**: Fast loading and responsive design
3. **Accessibility**: WCAG 2.1 AA compliant
4. **Consistency**: Unified design language across all pages
5. **Developer Focus**: Code examples and interactive demos
